# AI in Oncologic CT and MRI: Bibliometric Analysis (1997–2024)

R scripts (bibliometrix + igraph) for the study "Global trends on artificial 
intelligence in oncological applications of CT and MRI: a bibliometric 
analysis (1997–2024)".

## Contents
- `01_data_preparation.R` — data import, deduplication, metadata extraction.
- `02_data_analysis.R` — performance analysis, MCA, co-citation network.

## Data
Raw Scopus export not redistributed due to licensing restrictions.

## Citation
Celestino MS et al. Global trends on artificial intelligence in oncological 
applications of CT and MRI: a bibliometric analysis (1997–2024). 
Manuscript in preparation, 2026.
